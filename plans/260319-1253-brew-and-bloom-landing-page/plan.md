# Brew & Bloom Cafe Landing Page

## Overview
Single-page responsive landing site for a cozy cafe featuring modern design with coffee brown, green, cream palette.

## Tech Stack
- HTML5/CSS3 (vanilla or Tailwind CSS)
- Vanilla JS (minimal interactivity)
- Mobile-first responsive design

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| brown-dark | #4A3728 | Primary text, headers |
| brown-light | #8B7355 | Secondary elements |
| green-accent | #5D7B6F | CTAs, accents |
| cream | #F5F1EB | Backgrounds |
| white | #FFFFFF | Cards, contrast |
